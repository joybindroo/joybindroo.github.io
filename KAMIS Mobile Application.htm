KAMIS Mobile Application

         KAMIS mobile app is created using React Native framework. React Native is an open-source mobile app development technology used for building seamless cross-platform applications. It provides more efficient code sharing across mobile app platforms and the Web.

         KAMIS mobile app is available for use on Android mobile devices and can be downloaded from Google Play Store App. Link to install KAMIS app is also available on the login page of KAMIS web-portal (www.ka-mis.org).

         Currently block level users can use KAMIS app for data entry of cases, update of pending treatment details of cases, entry of VL and PKDL suspects, key-informants, data of concurrent monitoring of IRS. Forms in application are based on the forms available in web-portal.

         Application also has offline data entry feature and helps users in capturing data when Internet connection is unavailable. Data can be submitted later when Internet connection is available.
